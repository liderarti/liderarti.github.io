Treinamento técnico para certificação MCSA Windows 10 (exames 70-697 e 70-698).

Seja um especialista em Windows 10 e saia na frente no mercado de trabalho com a certificação MCSA Windows 10.

Nosso treinamento possibilita o conhecimento necessário e as habilidades do candidato para criar identidades sólidas, proteção de conteúdo (proteção contra perda de dados), política de gerenciamento de dispositivos móveis, virtualização com Hyper-V, gerenciamento de aplicativos usando o Windows Store.

Além do conhecimento técnico, o aluno que completar este treinamento terá as seguintes capacidades:

  • Conhecimento com administração, manutenção e solução de problemas de área de trabalho do Windows
  • Experiência básica e entendimento das tecnologias de rede do Windows
  • Conhecimento básico de Active Directory e Microsoft Intune
