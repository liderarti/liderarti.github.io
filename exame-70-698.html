Treinamento para certificação Windows 10

Treinamento prático preparatório para o exame de certificação 70-698 - Instalação e Configuração Windows 10.

Descrição

Este treinamento visa preparar o aluno a obter o conhecimento necessário para realizar os exames de certificação 70-698 e MD-100. Este curso engloba os seguintes assuntos:

Conteúdo

Setup do curso de Windows 10

Carga Horária

40 horas

Preço

R$ 499,00 R$ 399,00

Intrutor(es)
  • Antônio Carlos Costa Andrade (AC)
  • Aluísio Augusto Ferreira
Implementar o Windows
Preparar para os requisitos de instalação
Determinar os requisitos de hardware e compatibilidade; escolher entre uma atualização e uma instalação limpa; determinar edições apropriadas de acordo com o tipo de dispositivo; determinar os requisitos para recursos específicos, como Hyper-V, Cortana, Miracast, cartões inteligentes virtuais e Secure Boot; determinar e criar a mídia de instalação apropriada; configurar a prontidão de atualização;
Instalar o Windows
Executar instalações limpas; atualizar usando a mídia de instalação; configurar cenários de inicialização nativos; identificar caminhos de atualização válidos; migrar das versões anteriores do Windows; instalar no disco rígido virtual (VHD); inicialização da VHD; instalar no USB inicializável; instalar recursos adicionais do Windows; configurar o Windows para suporte regional e linguístico adicional;
Configurar dispositivos e drivers de dispositivos
Instalar, atualizar, desativar e reverter os drivers; resolver problemas do driver; ajustar as configurações do driver, incluindo drivers assinados e não assinados; gerenciar pacotes de drivers; baixar e importar pacotes de driver; usar a ferramenta de Gerenciamento e manutenção de imagens de implantação (DISM) para adicionar pacotes;
Realizar a configuração pós-instalação
Configurar e personalizar o menu Iniciar, a área de trabalho, a barra de tarefas e as configurações de notificação de acordo com o tipo de dispositivo; configurar as opções de acessibilidade; configurar a Cortana; configurar o Microsoft Edge; configurar o Internet Explorer; configurar o cliente Hyper-V; configurar as configurações de energia;
Implementar o Windows em um ambiente corporativo
Provisão com a ferramenta Windows Configuration Designer; implementar a ativação baseada no Active Directory; implementar a ativação de volume usando um serviço de gerenciamento de chaves (KMS); consultar e configurar estados de ativação usando a linha de comando; configurar o Active Directory, incluindo políticas de grupo; configurar e otimizar o controle de conta de usuário (UAC);
Configurar e dar suporte aos principais serviços
Configurar rede
Configurar e dar suporte às configurações de rede IPv4 e IPv6; configurar a resolução do nome; conectar-se a uma rede; configurar locais de rede; configurar o Firewall do Windows; configurar o Firewall do Windows com segurança avançada; configurar a descoberta de rede; configurar as configurações do Wi-Fi; configurar o Wi-Fi Direct; solucionar problemas de rede; configurar a VPN, como VPN desencadeada por aplicativo, filtros de tráfego e VPN de bloqueio; configurar IPsec; configurar o acesso direto;
Configurar armazenamento
Configurar discos, volumes e opções do sistema de arquivos usando o Gerenciamento de disco e o Windows PowerShell; criar e configurar VHDs; configurar dispositivos removíveis; criar e configurar espaços de armazenamento; solucionar problemas de armazenamento e dispositivos removíveis;
Configurar acesso aos dados e uso
Configurar o compartilhamento de arquivos e impressoras e conexões do Grupo doméstico; configurar o compartilhamento de pasta, pastas públicas e OneDrive; configurar as permissões do sistema de arquivos; configurar o uso do OneDrive, incluindo arquivos por demanda; solucionar problemas de acesso e uso de dados;
Implementar aplicativos
Configurar os aplicativos de área de trabalho; configurar as opções de inicialização; configurar os recursos do Windows; configurar a loja do Windows; implementar aplicativos do Windows Store; implementar o Windows Store for Business; implementar o Windows Store for Education; pacotes de provisão; criar pacotes; usar ferramentas de implantação; usar o Windows Assessment and Deployment Kit (ADK);
Configurar gerenciamento remoto
Escolher as ferramentas de gerenciamento remoto apropriadas; configurar configurações de gerenciamento remoto; modificar as configurações de forma remota usando o Microsoft Management Console (MMC) ou o Windows PowerShell; configurar a assistência remota, incluindo o Easy Connect; configurar o Remote Desktop; configurar o PowerShell remoto;
Gerenciar e manter o Windows
Configurar atualizações
Configurar as opções do Windows Update; implementar cenários Insider Preview, Branch Atual (CB), Branch Atual para Negócios (CBB) e Branch de Serviços de Longo Prazo (LTSB); gerenciar o histórico de atualizações; reverter atualizações; atualizar os aplicativos da Windows Store;
Monitorar o Windows
Configurar e analisar os logs do Event Viewer; configurar inscrições de eventos; monitorar o desempenho usando o Gerenciador de Tarefas; Monitorar o desempenho usando o Monitor de recursos; monitorar o desempenho usando o Monitor de desempenho e conjuntos de coletores de dados; monitorar recursos do sistema; monitorar e gerenciar impressoras; configurar as opções de indexação; gerenciar a segurança do cliente usando o Windows Defender; avaliar a estabilidade do sistema usando o Monitor de Confiabilidade; solucionar problemas de desempenho; gerenciar a segurança usando o Windows Defender Security Center; configurar o Windows Defender Advanced Threat Detection;
Configurar a recuperação do sistema e dos dados
Configurar uma unidade de recuperação; configurar uma restauração do sistema; executar uma atualização ou reciclagem; executar uma reversão do driver; configurar os pontos de restauração; resolver problemas de hardware e dispositivos; interpretar dados do Gerenciador de Dispositivos; restaurar versões anteriores de arquivos e pastas; configurar o histórico de arquivos; recuperar arquivos do OneDrive; usar o Backup e a Restauração do Windows; executar um backup e restaurar com o WBAdmin; executar as operações de recuperação usando o Windows Recovery;
Configurar autorização e autenticação
Configurar o Microsoft Passport; configurar senhas de imagem e dados biométricos; configurar grupos de trabalho; configurar configurações de domínio; configurar o ingresso no domínio do Azure AD; configurar as configurações do Homegroup; configurar o Credential Manager; configurar as contas locais; configurar as contas Microsoft; configurar o Registro de dispositivos; configurar o Windows Hello for Business; configurar o Device Guard; configurar o Credential Guard; configurar o Device Health Attestation (Atestado de Saúde do Dispositivo); configurar o comportamento do UAC; configurar o bloqueio dinâmico;
Configurar ferramentas de gerenciamento avançadas
Configurar serviços; configurar o Gerenciador de dispositivos; configurar e usar o MMC; configurar o Task Scheduler; configurar a automação das tarefas de gerenciamento usando o Windows PowerShell; converter objetos de Política de Grupo em políticas MDM usando a ferramenta de Análise de Migração MDM.
